Boundary Extraction for 3D binary image by Distance Transformation

A. Imiya

This program extracts boundary of 3D binary images. In the first step, we can obtain coordinate data which are centers of voxels of inner and outer side of the boundaries. In the second step, we assume that the common boundary surfaces of outer and inner voxels are the interface of 3D object. List of files and repository:
  • alg_2d.ps: description of the 2D boundary extraction algorithm
  • alg_3d.ps: description of the 3D boundary extraction algorithm
  • boundary.c: main C code
  • example: 2D and 3D examples
  • read_me.txt: Read this file first
